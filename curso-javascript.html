Curso de JavaScript Básico

Bem-vindo ao curso de JavaScript básico! Aqui você vai aprender os fundamentos da linguagem de programação mais popular da web.

O que é JavaScript?

JavaScript é uma linguagem de programação que permite adicionar interatividade e dinamismo a páginas web. É uma linguagem interpretada, ou seja, não precisa ser compilada antes de ser executada.

Pré-requisitos

Para acompanhar este curso, é necessário ter conhecimentos básicos de HTML e CSS. Se você ainda não tem esses conhecimentos, recomendamos que faça nosso curso de HTML e CSS antes de começar este.
