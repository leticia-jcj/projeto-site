Bem-vindo ao curso de HTML e CSS básico

Neste curso você aprenderá os conceitos básicos de HTML e CSS para poder criar páginas web simples. Ao final do curso, você será capaz de:

  • Criar uma página web usando HTML
  • Aplicar estilos a uma página web usando CSS
  • Criar links e imagens em uma página web
  • Criar tabelas e formulários em uma página web

Sobre o curso

O curso é dividido em quatro módulos, cada um com duração de uma semana. Cada módulo é composto por aulas em vídeo e exercícios práticos. O curso é totalmente online e você pode acessá-lo a qualquer momento, de qualquer lugar.

Instrutor
